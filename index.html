Recursive Portals


Description

Engineered a C++/DirectX 11 renderer to replicate and explore the spatial rendering and portal-traversal techniques found in Valve's Portal series.

Github

Role-Specific Tasks

  • Leveraged DirectX math libraries to compute:
    1. The view matrix of the virtual camera.
    2. The position of the camera as seen through the out portal.
    3. The oblique clipped projection matrix. This ensures that the near plane is aligned with the plane the portal is on.
  • Read/Write portals to the stencil buffer to accomplish recursive rendering.

UFO's Don't Land


Description

UFO's Don't Land is an anagram for the 2021 Global Game Jam's theme "Lost and Found". In this game your character was killed in an intergalactic space accident involving an alien UFO ship. You awake in a procedurally generated dungeon filled with alien monsters and must utilize objects in your environment to slay these enemies. Get to the end of the dungeon and you may just find yourself a second chance at life.

Github

Role-Specific Tasks

  • Wrote a script to procedurally generate the layout of the dungeon where each floor gets progressively more complex, all the while guaranteeing a final boss room.
  • Created spawn points for items and enemy locations, and implemented a rarity system that affects their probability of appearing.
  • Designed a fluent camera system to support multiple players on a single screen, and transitioning from one room to the next.

C-Sim


Description

C-Sim is a covid data visualizer and simulator. It takes live data from the CDC and uses it to simulate the spread of covid, taking various factors into account. The users can change numerous variables that affect the spread of covid and view data on the epidemic.

Click here to play.

Role-Specific Tasks

  • Randomly generated starting positions for homes within the bounds of the simulation, and the number of residents per household as well as their age, and gender.
  • Programmed the autonomous agents to move between their house and other points of interest, and social distancing based on the various input.
  • Created hospitals that serve as testing sites. If tested positive, they along with the rest of their household must return home to quarentine until they have all recovered.

Gobblet

Description

This initially started as a class project for RIT's Intro to Game Web Tech. It slowly turned into a passion project due to my interest in game theory and studying various algorithms associated with it. I implemented the Minimax algorithm for optimal decision making and used alpha-beta pruning to increase performance at run time. The AI maxizes each play by observing all possible outcomes several turns down the road and picking the move that results in the best outcome.

    • GSR Data Extraction

    Description

    Worked with a psychology professor at Suny Geneseo to help increase efficiency of data collection and processing. Subjects in this experiment are asked to place monetary bets on card games, while every tenth of a second their electrodermal activity is measured.
    The hypothesis is as follows: if there is a large flux in electrodermal activity (physiological reactivity), then that correlates to a higher stress level and an overall worse performance.

    Github

    Role-Specific Tasks

    • I created a Windows Form Application in C# that parses thousands of lines of electrodermal activity from an excel file and calculates the subjects physiological reactivity in 10 second intervals.
    • Data is processed neatly into an excel file for easy viewing.
